INSTRUMENTS, GHOST PLAY Documentation
Describes INSTRUMENTS, Ghost Play - Autonomous Violin-Player Imitation Devices source code written by Qt/C++.

Ghost Play is an autonomous viloin-playing device that instantly imitate human playing violin in real time.

This program will be dynamic linked to ppci7443.dll that controls ppci7443 motion controller board provided by Nippon Pulse Motor K.K / Nippon Pulse America, Inc. (NPM).
The DLL is NOT included, see http://nipponpulse.com/ for more information about the board.
This program also will be dynamic linked to phidget21.dll that controls phidgets. see http://www.phidgets.com/
for more information.

Without the DLLs, you may build this code safely, simulate this device, create score, but you can not operate device without the DLLs.
Dependency

This program depends on following libraries:
  • libfftw-3.3 ( FFTW: http://www.fftw.org/ )
  • Qt4.5 ( Qt: http://qt.nokia.com/ )
  • Phidget21.dll ( Phidget: http://www.phidgets.com/ )
  • ppci7443.dll ( PPCI7443 Motion Controller Board : http://nipponpulse.com/ )
